研究经历

视空工作记忆与视觉组块策略研究

认知心理学实验设计与数据分析

主导设计并实施了关于视空工作记忆（Visuospatial Working Memory）的心理学实验，重点探究视觉组块的形成与提取机制。负责清洗与整理多维度的实验数据，并运用协方差分析 (ANCOVA) 及马氏距离 (Mahalanobis distance) 等统计方法剔除异常值并进行深度建模，确保实验结论的严谨性。

实验设计 数据建模 统计分析

开发项目

在线心理学测试与数据采集平台

独立开发部署的 Web 端心理测试环境。平台内置 Corsi 敲击任务 (Corsi block-tapping task) 与 Raven 瑞文推理测验矩阵。通过脚本实现了防作弊的全屏限制、高精度的鼠标点击坐标与时间参数采集，保障了被试数据的有效性与精确度。

JavaScript Python 全栈部署

歌词语法自动解析与同步系统

基于 WebNowPlaying 接口开发的语言学习辅助原型工具。系统能够将歌曲音频与歌词文本精准同步，并通过自动化脚本对韩语歌词进行逐词拆解、词汇提取与语法点高亮解析，为跨语种学习提供了创新的交互体验。
